Period 3 - Consumer Math

This course teaches real-life application of math skills
that will be needed after graduation by using the AGS
Consumer Math textbook. Students focus on improving
their problem solving skills with the use of a
calculator. This course builds skills needed for the
Compass test (entrance exam to Bellevue College,
Renton Technical College and the Lake Washington
Institute of Technology).

Projects:

  • Create A Budget: Find A Job Powerpoint
  • Create A Budget: Find A Job Worksheet

Four-Point Grading System:

In order to earn all four points, the work must be on time
with every problem attempted and all work shown

  • Halfway completed = 2/4
  • Additional point subtracted if the assignment is late
